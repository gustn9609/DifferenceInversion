Abstract

We present Difference Inversion, the first model-agnostic inversion method that isolates and applies only the precise difference between an image pair {A, A′} onto a new query image B, without introducing unwanted artifacts. By interpolating between image and text deltas using token consistency, our method faithfully captures the transformation A → A′ and transfers it onto B to generate a plausible B′. Unlike previous baselines, Difference Inversion can generalize across different diffusion backbones (e.g., SDXL), enabling new applications in image editing and many other analogy-based computer vision tasks.
